# Lam3a App

A comprehensive Flutter-based mobile application that connects **vehicle owners** with professional **service providers** and **car washers**. Lam3a simplifies the process of booking vehicle services, managing your garage, and connecting with trusted automotive service professionals.

**For Vehicle Owners:** Easily manage your vehicle fleet, book car services including car washing and maintenance, select from verified service providers, schedule convenient time slots, and track all your service requests in one place.

**For Service Providers & Car Washers:** Manage your availability, accept service requests from vehicle owners, track your bookings, and grow your automotive service business with a professional platform built for efficiency and customer satisfaction.

Built with Flutter for iOS and Android, Lam3a delivers a seamless, modern experience for both vehicle owners and service professionals.

## Getting Started

### Prerequisites

- Flutter SDK (3.9.2 or higher)
- Dart SDK
- Git

### Running the App

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abdullaalsabti/Lam3aApp.git
   cd Lam3aApp
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. **Connect your mobile device** or start an emulator:
   - For Android: Connect your Android device via USB (with USB debugging enabled) or start an Android emulator
   - For iOS: Connect your iPhone via USB or start an iOS simulator (macOS only)
   - Verify the device is connected by running: `flutter devices`

4. Run the app:
   ```bash
   flutter run
   ```

5. **Wake up the backend service** by visiting: https://lam3a-backend.onrender.com/
   - This activates the backend API service (required for the app to function properly)
